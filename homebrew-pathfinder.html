Adding Homebrew Choices to Quilvyn's Pathfider Rule Set

The Pathfinder rule set allows you to add homebrew choices to all of the same selections discussed in the SRD v3.5 Homebrew Examples document. The only change to the dialogs shown for SRD v3.5 is that Pathfinder skills do not have synergies. In addition, Pathfinder also allows you to add homebrew factions and traits.

Factions

Factions add new goals to character motivation in Pathfinder Society play. Adding a homebrew faction requires only specifying the name.

The example shows entries for a Galtan faction—perhaps a group whose goal is to export revolution to the rest of the Inner Sea region.

Traits

Traits describe small customizations that help to expand character backgrounds and motivations. When entering a homebrew trait, Quilvyn provides entry fields for the trait type (Basic, Campaign, Faction, Race, Regional, or Religion), and subtype.

The example shows entries for the Child Of Infamy campaign trait from the Council of Thieves Player's Guide and the entries for the corresponding feature.
